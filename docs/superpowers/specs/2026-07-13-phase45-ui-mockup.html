Engine running too lean (bank 1)
W203-specific Confidence: Confirmed
What's happening
The engine is getting more air than the fuel math expects, so the computer keeps adding fuel to compensate.
Can I keep driving?
Yes, for now — but sustained lean running can slowly damage the catalyst.
Check first
  1. Partial-load breather hoses (THE M271 weak spot)
  2. Intake leaks after the MAF
  3. MAF sensor readings
  4. Fuel pressure
